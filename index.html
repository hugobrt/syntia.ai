Politique de Confidentialité

Dernière mise à jour : 03 Février 2026

Chez Syntia, la protection de vos données est primordiale. Voici comment nous traitons vos informations :

1. Données Techniques (IDs)

Pour assurer le fonctionnement du Bot, nous stockons uniquement : Server ID, Channel IDs, Admin User IDs.

2. Clés API & Sécurité

Les clés API (Groq, Llama) sont stockées de manière chiffrée dans les variables d'environnement (Render Cloud). Aucune clé n'est exposée.

3. Non-Rétention

Syntia n'enregistre aucune conversation. Tout est traité en RAM.
