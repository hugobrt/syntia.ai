Politique de Confidentialité

Dernière mise à jour : 03 Février 2026

Chez Syntia, la protection de vos données est primordiale. Voici comment nous traitons vos informations :

1. Données Techniques (IDs)

Pour assurer le fonctionnement du Bot (modération, logs, gestion de salon), nous stockons uniquement les identifiants techniques nécessaires :

  • Server ID (Guild ID) : Pour identifier votre communauté.
  • Channel IDs : Pour savoir où envoyer les messages.
  • Admin User IDs : Pour vérifier les permissions d'accès au panel.

2. Clés API & Sécurité

Les clés API utilisées pour connecter le bot à l'intelligence artificielle (Groq, Llama) et aux services externes ne sont jamais stockées dans le code source public ni accessibles côté client. Elles sont sécurisées via les variables d'environnement de notre hébergeur (Render Cloud), garantissant une étanchéité totale.

3. Non-Rétention des Messages

Syntia n'enregistre aucune conversation. Les messages envoyés à l'IA sont traités en mémoire vive (RAM) le temps de générer la réponse, puis sont immédiatement détruits.

Conditions Générales d'Utilisation

1. Acceptation

En invitant le bot Syntia ou en rejoignant notre serveur Discord, vous acceptez pleinement les présentes conditions ainsi que les Conditions d'Utilisation de Discord.

2. Usage Responsable

Le module d'administration (Panel Infinity) donne des pouvoirs de modération étendus. L'utilisateur est seul responsable des commandes exécutées (ban, kick, clear). Syntia décline toute responsabilité en cas de mauvaise configuration ou d'abus par vos administrateurs.

3. Avertissement IA

Les fonctionnalités liées à l'IA (analyse de marché, conseils business) sont fournies à titre indicatif. Llama 3.1 est un modèle probabiliste : ses réponses ne constituent en aucun cas un conseil financier officiel. Vous restez maître de vos investissements.
